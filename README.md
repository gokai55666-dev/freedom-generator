# Freedom Generator

Cloud-enhanced AI generation with maximum capability analysis.

## Deploy to Render

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy)

Or manually:
1. Fork this repo
2. Create new Web Service on Render
3. Connect this repo
4. Render auto-detects `render.yaml`

## Perchance Frontend

Copy `perchance/freedom-generator.txt` into perchance.org

## Android Connection

Install Tampermonkey script from `perchance/freedom-connector.user.js`
